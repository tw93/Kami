Electron ships a whole browser with every app. Pake takes the opposite bet: lean on the system webview and write the shell in Rust, so the same web app becomes a desktop build that is roughly 40× smaller and starts in well under a second.

Why Pake

Most "wrap my site as an app" tools trade size and speed for convenience. Pake refuses the trade.

  • Bundles around 5 MB, against 100 MB+ for an equivalent Electron build.
  • Native shortcuts, immersive window, and ad-blocking baked in.
  • One CLI command, or a no-code GitHub Action, to ship.

How it works

A Tauri + Rust core drives the platform webview directly, with thin per-OS adapters.

  • Rust shell handles windows, tray, and injection.
  • System webview renders the page, no Chromium copy.
  • One config maps URL, icon, and shortcuts to a build.
