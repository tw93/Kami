Changelog

Breaking Changes

  1. Breaking Template registry moved to scripts/shared.py. Custom forks that patched build.py directly must re-point at the new HTML_TEMPLATES map.
  2. Breaking Commercial CJK fonts are no longer bundled in dist/kami.zip. Run scripts/ensure-fonts.sh once to restore local preview paths.

Features

  1. Added the Landing Page template with five deployment companions (vercel, sitemap, robots, llms, llms-full).
  2. Slides now ship three rendering paths: WeasyPrint HTML to PDF, editable PPTX via python-pptx, and a Markdown-first Marp variant.
  3. Pygments syntax highlighting for code blocks, with a clean monochrome fallback when it is not installed.
  4. Resume gained a dense variant that keeps five-plus project entries inside a strict two-page bound.

Fixes

  1. Tightened resume project spacing so company blocks no longer crowd against each other on page one.
  2. Resolved CJK parenthesis wrapping that broke line rhythm in Chinese and Korean templates.
  3. Cross-template token drift is now caught by build.py --check before packaging.
